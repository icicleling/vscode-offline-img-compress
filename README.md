# offline-img-compress

This is an offline image compression extension, a simple wrapper for [sharp](https://github.com/lovell/sharp), providing pure local compression.

## Usage

Right-click on the file and select "Compression File". The compressed file will be generated with the name `[filename]-compressed.[ext]`.

For example, `foo.png` -> `foo-compressed.png`.
